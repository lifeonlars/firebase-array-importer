Simple Firebase Import Script

Pushes an array of objects into Firebase one by one and generates individiual push keys for each new object.

  1. Make sure you've added your Firebase config settings in scripts/main.js
  2. Choose the Firebase database reference where you want the array to be pushed e.g. /products or / for root.
  3. Select your local JSON file containing the array and click Import Now
Select file and Firebase DB path

JSON file needs to contain a root array.
